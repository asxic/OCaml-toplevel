# OCaml-toplevel
OCaml toplevel running in the browser

Live version here : https://math-gout.github.io/OCaml-toplevel/index.html

## Supported features
- Syntax highlighting
- Autocompletion
- Standard OCaml library supported
- Graphics library supported
- Open/Save .ml file
